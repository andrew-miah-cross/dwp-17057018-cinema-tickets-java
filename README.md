# Coding Test for DWP

See [TASK.md for details](TASK.md) of the coding test set by DWP.

#### Issues

* No notion of 'basket' reservations to prevent errors when there are no seats left to sell.
* No support for the reservation service being down, refunds etc., i.e. money is taken but no seat is provided.
* Since there's no support for refunds or cancellation, ticket purchase/reservation order is irrelevant.
* Price is in GBP and specified as integer where float or pence might be preferred.
